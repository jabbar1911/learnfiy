Learn Python – From Basics to Real Projects

Master Python programming from the fundamentals to real-world projects. Learn variables, loops, functions, OOP, file handling, and build practical applications. Perfect for beginners preparing for coding, development, and problem-solving.

Course Introduction

This Learnify Python course teaches you Python from the absolute basics to real-world applications. Start with variables and data types, then move into loops, functions, data structures, OOP, file handling, modules, and mini-projects. Perfect for beginners who want to learn programming, automation, backend development, or prepare for coding interviews.

Why Learn Python?

  • Beginner-Friendly — Simple, readable syntax makes Python easy for new learners.
  • Most Versatile Language — Used in web apps, automation, data science, AI, cybersecurity, and more.
  • High Demand — Python developers are highly sought after in companies worldwide.
  • Strong Community — Huge ecosystem of libraries, tutorials, forums, and community support.

What You'll Learn

  • Python Basics — Variables, data types, input/output, and core syntax.
  • Control Flow — if-else, loops, logical conditions, and decision making.
  • Functions — Defining functions, arguments, return values, and modular coding.
  • Data Structures — Lists, tuples, sets, dictionaries, and when to use each.
  • Object-Oriented Programming (OOP) — Classes, objects, inheritance, polymorphism, and encapsulation.
  • File Handling — Reading and writing files, working with text, CSV, and JSON.
  • Error Handling — Try/except, exceptions, and writing safe programs.
  • Modules & Packages — Importing libraries, using built-in modules, and creating your own.
  • Python Libraries — Introduction to important libraries like NumPy, Pandas, Matplotlib (basic level).
  • Working With APIs — Fetching data from web APIs and basic JSON handling.
  • Automation Basics — Using Python scripts to automate simple tasks.
  • Mini Projects — Build small programs using all concepts learned.

  • What Are Variables?
    Variables store data in memory. You assign values using the = operator. 
    x = 10
name = "Learnify"
  • Variable Naming Rules:
    • Must start with a letter or underscore
    • Cannot start with a number
    • No special characters (%, $, @, etc.)
    • Case-sensitive (age ≠ Age)
    • Use meaningful names
  • Basic Data Types in Python:
    • int — Whole numbers (10, -4)
    • float — Decimal numbers (3.14, 9.5)
    • str — Text ("Hello")
    • bool — True or False
    Example: 
    a = 10
b = 3.5
c = "Learnify"

    Convert one data type to another using functions:
    Function Converts To Example Output

    bool() Boolean bool(1) True

  • if Statement:
    Used to run a block of code only when a condition is true. 
    age = 18
if age >= 18:
    print("You are eligible")
  • if-else Statement:
    Executes one block if condition is true, otherwise runs the else block. 
    num = 5
if num % 2 == 0:
    print("Even")
else:
    print("Odd")
  • elif Ladder:
    Used when multiple conditions must be checked one after another. 
    marks = 72

    print("B Grade")

    print("C Grade")
else:
    print("D Grade")

    An if statement inside another if block. 
    username = "admin"

  • while Loop:
    Repeats a block of code as long as the condition remains true. 
    i = 1
while i <= 5:
    print(i)
    i += 1
  • for Loop:
    Used to loop through a sequence like range, string, list, tuple, etc. 
    for i in range(1, 6):
    print(i)
    Looping through a string: 
    for ch in "Learnify":
    print(ch)
  • Loop Control Statements:
    • break — Stops the loop immediately
    • continue — Skips the current iteration
    • pass — Placeholder (does nothing)
    Examples: 
    # break
for i in range(1, 10):
    if i == 5:
        break
    print(i)
    # continue
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
    # pass
for i in range(5):
    pass  # Used to avoid errors in empty blocks
  • range() Function:
    Used to generate a sequence of numbers.
    Syntax Description Example
    range(n) 0 to n-1 range(5) → 0 1 2 3 4
    range(start, end) start to end-1 range(2, 6) → 2 3 4 5
    range(start, end, step) Custom increment range(1, 10, 2) → 1 3 5 7 9

  • Recursion Basics:
    A function calling itself to solve problems like factorial, Fibonacci, etc. 
    def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))   # 120

nums.reverse()     # [10, 5, 4, 1]
    Slicing example:

  • Set Basics & Operations:
    Sets store unique values (no duplicates). 
    myset = {1, 2, 3, 3}

    Set operations: 
    a = {1, 2, 3}
b = {3, 4, 5}

print(a.intersection(b)) # {3}
print(a.difference(b))   # {1, 2}

  • What is OOP?

    OOP (Object-Oriented Programming) is a programming method where we structure programs using objects. Objects contain data (variables) and behavior (functions).

    Python supports OOP and makes programs easier to read, reuse, and maintain.

  • Class & Object:

    Class → A blueprint/model to create objects.

    Object → A real instance created from a class.

    class Student:
    name = "Rahul"     # Attribute

s1 = Student()         # Creating object
print(s1.name)
  • Constructors (__init__):

    The constructor is a special method that runs automatically when an object is created.

    class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

s1 = Student("Rahul", 20)
print(s1.name, s1.age)
  • Methods & Attributes:

    Attributes → Variables inside a class.

    Methods → Functions inside a class.

    class Calculator:
    def add(self, a, b):     # Method
        return a + b

c = Calculator()
print(c.add(5, 3))
  • Major OOP Concepts (4 Pillars):
    • 1. Encapsulation
      Binding data + methods together and restricting access using _ or __. 
      class Person:
    def __init__(self):
        self.__salary = 50000   # Private variable

p = Person()
# print(p.__salary)  # Error: cannot access private variable
    • 2. Inheritance
      One class can inherit properties from another class. 
      class Animal:
    def sound(self):
        print("Some sound")

class Dog(Animal):      # Inheriting Animal
    pass

d = Dog()
d.sound()
    • 3. Polymorphism
      Same function name, different implementation. 
      class Bird:
    def fly(self):
        print("Bird is flying")

class Penguin:
    def fly(self):
        print("Penguins cannot fly")

def check(obj):
    obj.fly()

check(Bird())
check(Penguin())
    • 4. Abstraction
      Hiding complex details and showing only important parts. 
      from abc import ABC, abstractmethod

class Car(ABC):
    @abstractmethod
    def start(self):
        pass

class Tesla(Car):
    def start(self):
        print("Tesla started")

t = Tesla()
t.start()
  • Real-World Example: Student Management 
    class Student:
    def __init__(self, name, marks):
        self.name = name
        self.marks = marks

    def display(self):
        print(self.name, self.marks)

s = Student("Jabbar", 90)
s.display()

  • 1. math Library
    Used for mathematical calculations such as square roots, powers, trigonometry, constants. 
    import math

print(math.sqrt(25))      # 5.0
print(math.pow(2, 3))     # 8.0
print(math.pi)            # 3.14159
print(math.floor(4.9))    # 4
print(math.ceil(4.1))     # 5
  • 2. random Library
    Used to generate random numbers — useful in games, testing, simulations. 
    import random

print(random.randint(1, 10))     # Random number from 1–10
print(random.choice(["Python", "HTML", "CSS"]))  # Random choice
print(random.random())           # Random float (0 to 1)
  • 3. datetime Library
    Helps work with dates and time (current date, formatting, adding days, etc.) 
    from datetime import datetime, timedelta

now = datetime.now()
print(now)                          # Current date & time
print(now.strftime("%d-%m-%Y"))     # Format date

future = now + timedelta(days=7)
print(future)                       # Date after 7 days
  • 4. os & sys Libraries
    These let you interact with the operating system.

    os → working with files, folders, paths

    sys → system-level information

    import os
import sys

print(os.getcwd())        # Current working directory
print(os.listdir())       # List files/folders
print(sys.version)        # Python version

    • Use list comprehensions when possible

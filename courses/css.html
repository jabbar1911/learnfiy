Learn CSS – Style, Design & Build Beautiful Websites

Master CSS from basics to professional-level styling. Learn how to control colors, layouts, typography, responsiveness, animations, and build modern web designs. Perfect for beginners who want to build clean, attractive webpages and improve frontend skills.

Course Introduction

This Learnify CSS course teaches you how to style webpages from scratch. You will learn everything from basic colors and fonts to advanced layouts like Flexbox and Grid, media queries, animations, shadows, transitions, responsive design, and UI components.

Why Learn CSS?

  • Brings Websites to Life — CSS controls colors, spacing, layout, and the entire look of a site.
  • Essential for Frontend Development — Every professional frontend job requires CSS mastery.
  • Responsive Web Design — Learn how to make websites work on mobile, tablet, and desktop.
  • Modern UI Skills — Flexbox, Grid, animations, shadows, transitions, etc.
  • Works With Every Framework — React, Bootstrap, Tailwind, Angular all rely on CSS fundamentals.

What You'll Learn

  • CSS Basics: Selectors, properties, colors, fonts, spacing.
  • Styling Text: Fonts, line-height, alignment, formatting.
  • Box Model: Margin, padding, borders, sizing.
  • Flexbox: Modern layout system for aligning and arranging items.
  • Grid Layout: Advanced 2D layouts for complete web page design.
  • Positioning: Relative, absolute, fixed, sticky.
  • Backgrounds & Shadows: Gradients, patterns, box shadows.
  • Transitions & Animations: Smooth effects, hover animations, keyframes.
  • Responsive Design: Media queries, breakpoints, mobile-first design.
  • CSS Units: px, %, em, rem, vh/vw, fr.
  • Forms & Buttons: Styling inputs, buttons, UI components.
  • Variables & Advanced Concepts: CSS variables, calc(), clamp().
  • Complete Project: Build a fully responsive modern landing page.

Course Syllabus

  • What is CSS?
    CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, alignment, borders, backgrounds, and full layouts.
  • Ways to Use CSS:
    • Inline CSS: Inside the HTML tag 
      <h1 style="color:red;">Hello</h1>
    • Internal CSS: Inside <style> in the head 
      <style>
h1 { color: blue; }
</style>
    • External CSS (Recommended): In a separate .css file 
      /* style.css */
h1 {
  color: green;
}
      Linked using: 
      <link rel="stylesheet" href="style.css">
  • Basic Styling Example: 
    h1 {
  color: blue;
  font-size: 30px;
  font-weight: bold;
}
  • Common CSS Selectors: 
    /* Element Selector */
p { color: gray; }

/* Class Selector */
.title { font-size: 24px; }

/* ID Selector */
#main { background: yellow; }
  • CSS Units:
    • px — fixed pixel size
    • % — percentage of parent container
    • em — relative to parent font-size
    • rem — relative to root font-size
    • vh / vw — viewport height / width
    Example: 
    .box {
  width: 50%;
  font-size: 1.2rem;
  height: 50vh;
}

  • Basic Selectors:
    • Element Selector: p { color: blue; }
    • Class Selector: .title { font-size: 22px; }
    • ID Selector: #header { background: lightgray; }
  • Attribute Selectors:
    Used to target elements based on attributes and values. 
    input[type="text"] {
  border: 1px solid grey;
  padding: 6px;
}
  • CSS Combinators:
    Used to select elements based on their relationship in the HTML.
    • A B (Descendant): Selects all B inside A 
      div p { color: green; }
    • A > B (Child): Selects only direct children 
      ul > li { padding: 5px; }
    • A + B (Adjacent Sibling): Selects B that comes immediately after A 
      h1 + p { font-size: 18px; }
    • A ~ B (General Sibling): Selects all B siblings after A 
      h2 ~ p { color: gray; }
  • CSS Specificity:
    Determines which style wins when multiple rules apply.
    • Inline styles → Highest priority
    • ID selectors → #id
    • Class selectors → .class
    • Element selectors → h1
    Example: 
    #title { color: blue; }
.title { color: red; }
h1 { color: green; }
    Final color → blue (ID wins)

CSS Positioning controls *where* elements appear on the page. It is used for tooltips, modals, floating buttons, sticky headers, and advanced layouts.

  • Static — default flow
  • Relative — moves element relative to itself
  • Absolute — positioned relative to nearest parent
  • Fixed — stays fixed on scroll
  • Sticky — sticks when scrolled
  • z-index — controls layering
1. Static Position (Default)
Static Box 1
Static Box 2
.box {
  position: static;
}

Top/left/bottom/right do NOT work for static elements.

2. Relative Position
Moved using top + left 
.box {
  position: relative;
  top: 10px;
  left: 20px;
}
3. Absolute Position
Absolute Child

(Parent is relative → child positions inside parent) 

.parent { position: relative; }
.child  { position: absolute; }
4. Fixed Position
Fixed Button

Scroll the page — this button stays fixed.

.fixed-btn {
  position: fixed;
  bottom: 20px;
  right: 20px;
}
5. Sticky Position
I Stick When Scrolling

Scroll to see sticky effect

.header {
  position: sticky;
  top: 0;
}
6. z-index (Layer Control)
Behind (z-index:1)
Front (z-index:3)

.box2 { z-index: 1; }
7. Common Layout Patterns
  • Floating banners & popups (absolute + z-index)
  • Sticky headers (sticky)

Responsive Design ensures your website looks great on mobiles, tablets, and desktops. It uses flexible layouts, scalable text, fluid containers, and media queries.

  • Mobile-first design
  • Media Queries
  • Responsive units (% , rem , vw , vh , fr)
  • Responsive images
  • Responsive Flexbox & Grid layouts
Mobile-First Approach

Start designing for small screens → scale up for larger devices.

Mobile padding: 15px
Tablet/Desktop padding: 40px 
/* Base - Mobile */
.container {
    padding: 15px;
}

/* Tablets & larger */
@media (min-width: 768px) {
    .container {
        padding: 40px;
    }
}
Media Queries

Change layout based on screen size.

@media (max-width: 768px) {
    h1 {
        font-size: 22px;
    }
    .box {
        width: 100%;
    }
}
  • max-width → styles below width
  • min-width → styles above width
  • orientation → landscape / portrait
Responsive Images
Image scales automatically to fit its container. 
img {
    max-width: 100%;
    height: auto;
}
Responsive Grid Layout
1 column on mobile → 3 columns on tablet & desktop. 
.grid {
    display: grid;
    grid-template-columns: 1fr;
    gap: 20px;
}

@media (min-width: 768px) {
    .grid {
        grid-template-columns: repeat(3, 1fr);
    }
}

Perfect for product grids, cards, galleries.

Responsive Text
Text size adapts between 24px and 48px based on screen width. 
h1 {
    font-size: clamp(24px, 5vw, 48px);
}

clamp() helps create fluid, scalable typography.

CSS Variables allow you to store reusable values like colors, spacing, and fonts. Advanced CSS helps you build scalable, flexible, and modern UI designs.

  • Creating and using CSS Variables
  • Global variables with :root
  • Overriding variables
  • Dark mode using variables
  • calc() for mathematical styling
  • Advanced selectors: :nth-child, :not(), :is()
  • Advanced units: vh, vw, min(), max(), clamp()
CSS Variables (Custom Properties)
Define reusable values in :root to make your CSS cleaner. 
:root {
    --primary: #0d6efd;
    --radius: 8px;
    --padding: 20px;
}

.box {
    background: var(--primary);
    padding: var(--padding);
    border-radius: var(--radius);
}
Overriding Variables
:root {
    --theme-color: #0d6efd;
}

.dark-mode {
    --theme-color: #111;
}

.title {
    color: var(--theme-color);
}

Add class="dark-mode" to switch themes instantly.

Dark Mode With CSS Variables
:root {
    --bg: #ffffff;
    --text: #222;
}

.dark-mode {
    --bg: #000000;
    --text: #f5f5f5;
}

body {
    background: var(--bg);
    color: var(--text);
}
Using calc() for Dynamic Values
.box {
    width: calc(100% - 40px);
    padding: calc(10px + 1rem);
}
Advanced CSS Selectors
  • :nth-child() 
    li:nth-child(2) { color: red; }
  • :not() 
    button:not(.primary) { background:#eee; }
  • :is() 
    .content :is(h1, h2, h3) {
    margin-bottom: 10px;
}
Advanced CSS Units
  • vw → viewport width
  • vh → viewport height
  • min() → smallest value
  • max() → largest value
  • clamp() → responsive range
h1 {
    font-size: clamp(1.5rem, 5vw, 3rem);
}
